# Mon Projet
Cette application est un projet d'apprentissage. Elle doit me permettre d'appliquer les compétences aquises et développer mon sens de la recherche afin de trouver des solutions
par moi même. 

C'est une appilcation que je souhaite développer jusqu'à la production. Tous les domaines du développement web doivent être abordés.

La phase 1 consiste à réaliser cette application web du front au backend sans l'utilisation de l'IA.
La phase 2 consiste à réaliser la même maquette avec l'utilisation de l'IA. 

L'objectif doit être de comprendre les bases du dev afin de travailler avec les outils moderne.
